# subscribe-yun
自动订阅云盘分享

#声明<br><br>


#使用教程<br><br>
##下载docker镜像：docker push kivenray/subscribe-yun:tagname<br>

##环境变量配置<br>
###TG_BOT_TOKEN: TG机器人token<br>
###TG_BOT_USER_NAME: TG机器人名称<br>
###TG_BOT_CHAT_USER_ID:你的TG账号id<br>
###ALIYUN_REFRESH_TOKEN:阿里云盘的refresh_token<br>
